# Описание проекта
Проект представляет собой приложение-мотиватор, заставляющее меньше сидеть в "Бесполезных" программах/приложениях, играх и тому подобном, занимать пользователя "Полезной" деятельностью. Планируется, что программа будет предоставлять пользователю несколько типов работы с программой

Бывает время, когда необходимо выполнить ту или иную работу, решить некоторую проблему, но при этом отвлекают развлекающие факторы, по типу скроллинга ленты или игр. 
Мой проект поможет решить эту проблему.

Основными пользователями будут люди, которым сложно сосредоточиться на одном деле, но при этом необходимо выполнить ту или иную работу.
Таким образом получаем, что пользоваться этим проектом будут студенты, обучающиеся (или их родители, установившие приложение на телефон своему ребёнку), некоторые взрослые люди, работающие из дома
